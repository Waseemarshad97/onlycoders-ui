## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Index.js is our file where we initially load the components.

main.js is the main body component which have all the child components fo the UI
body components are stored in components folder
